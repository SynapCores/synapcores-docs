# Support

## Bug reports

File issues on the public release-mirror repository:
[github.com/SynapCores/synapcores-releases/issues](https://github.com/SynapCores/synapcores-releases/issues).

Please include:

- Output of `synapcores --version`
- Your OS / distro / kernel (`uname -a` + `cat /etc/os-release`)
- Steps to reproduce
- A minimal config or SQL example if relevant

Source-level reports (file paths, line numbers) aren't actionable
because the source isn't published; please describe the user-visible
behavior instead.

## Security disclosure

Found a vulnerability? Please **don't** file it as a public issue.
Email `security@synapcores.com` with:

- A description of the issue
- Reproduction steps
- The version affected (`synapcores --version`)
- Your timeline expectation (we aim to acknowledge within 48h)

We follow [coordinated disclosure](https://en.wikipedia.org/wiki/Coordinated_vulnerability_disclosure).
